### API functions 
**Disclaimer: This is the roadmap of the API "calls" that should be available for client apps for the very first version of Franklin-api (code name: Ashurbanipal)**

## Users
* Sign In and Sign Up (create account and all that stuff)
* Add book to shelf
    * Mark a book as read
    * Mark a book as BR (being read)
    * Mark a book as TBR (to be read)
* Get all users' books

## Books
